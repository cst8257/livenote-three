# Livenote Three

## Objective
Create database-driven notes application with authentication using Laravel and Livewire.

## Structure
This project has been broken into six parts. Each part will build onto the previous part and will introduce a new feature of Laravel or Livewire. 

### Part 1: Models and Databases
Part 1 will introduce Laravel models and working with databases including connecting, migrating, and seeding tables.

### Part 2: Blade Layouts
Part 2 will introduce Blade layouts and components for improved reusability and management of the code.

### Part 3: Livewire and Flux Components
Part 3 will introduce Livewire single file components and pages, which will be necessary for make the site more reactive. We will also look at FluxUI component library.

### Part 4: Actions, Data Binding, and Data Manipulation
Part 4 will add reactivity using the Livewire components and pages. 

### Part 5: Authentication and Validation
Part 5 will introduce Laravel authentication and validation. The ability to login and logout will be added to the web application.
